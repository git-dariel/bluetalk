2. COMPONENT RULES

---

Buttons:

- Follow defined padding
- Follow state transitions
- No custom shadows

Chat Bubble:

- Must match alignment rules
- Must match border radius rules
- Must reflect sender state

Device List:

- Must follow card spacing
- Must follow active selection highlight

---

3. RESPONSIVENESS

- Handle small screen
- Handle large screen
- Maintain layout integrity
- Avoid overflow clipping
